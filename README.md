CRA PWA React example on TypeScript and Stylus with i18next
===========================================================


Deploy
------

Установка зависимостей npm packages

    $ npm install

Запуск сервера для разработки
-----------------------------

    $ npm start

    http://localhost:3000/

Cборка
------

Сборка проекта в продакшен

    $ npm build

Тесты
-----

Запуск тестов

    $ npm test
